# APT Publishing Notes

This repository uses `.deb` artifacts generated by `cargo-deb`.

## Build `.deb`
```bash
./scripts/release/build-deb.sh
```

## Publish with aptly
```bash
./scripts/release/publish-apt.sh dist/<package>.deb stable
```

## Client install example
```bash
# Add your published apt repository endpoint and key first
sudo apt update
sudo apt install tidemark
```
